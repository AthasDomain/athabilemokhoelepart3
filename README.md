GreenWave Gurus
A responsive website for GreenWave Gurus, a Port Elizabeth based waste management and recycling initiative focused on environmental sustainability and community engagement.

## Project Overview
GreenWave Gurus provides waste drop-off points, recycling services, community cleanups, and environmental education in Gqeberha. This site is built for Part 2 of the project with interactive JavaScript features.

## Pages & Features
- Home: Hero banner, mission overview, "Did You Know?" facts, staggered animations
- About: Mission, vision, key environmental legislation with accordion sections
- Services: 7 service cards with expandable details and hover effects
- Enquiry: Contact form with real-time validation and FAQ accordion
- Contact: Contact details with copy-to-clipboard, social links, 3 waste drop-off locations with lazy-loaded Google Maps
- Gallery: Project photo gallery

## Tech Stack
- HTML5, CSS3
- Vanilla JavaScript - no libraries
- Google Maps Embed API
- IntersectionObserver for scroll animations

## How to Run Locally
1. Clone this repository
2. Open `index.html` in your browser
3. No build tools required
